# 🍽️ Python Restaurant Menu App

An interactive **Restaurant Menu application** built with **Python** and **Streamlit**. This app allows users to browse a menu, view item images, add items to their order, track the total, and complete the order in a seamless, user-friendly interface.

---

## 🛠 Features

- Dynamic menu display with images from **local files** or **image URLs**.  
- Real-time order tracking using **Streamlit session state**.  
- Clean, responsive layout using multiple columns for menu items.  
- “Add to Order” and “Finish Order” workflow with **total calculation**.  
- Works entirely in the browser — no backend setup required.  

---

## 💻 Technologies Used

- **Python 3.x**  
- **Streamlit** for interactive UI  
- **os module** for local file management
